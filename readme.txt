Para compilar o programa, fazer javac puzzle.java no terminal, estando na pasta que contem o ficheiro puzzle.java e todos os outros ficheiros .java correspondentes as classes do programa.
Para executa-lo, apos ter sido compilado, basta escrever java puzzle.
Sera pedido ao utilizador que insira primeiro apenas a configuracao inicial, e so depois a configuracao final, e nao ambos de uma vez.
Se existir caminho entre as duas configuracoes, o utilizador vai ver uma lista com as pesquisas que o trabalho disponibiliza. Aqui, pede-se que se introduza um unico numero na linha de comandos, correspondente a pesquisa pretendida.
